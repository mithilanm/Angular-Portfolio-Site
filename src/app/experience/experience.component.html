First to 20 Game

This application uses the Arduino board to play the classic first to 20 game. Here's how the game works. You as the user will compete with the computer to reach the number 20 first by only adding either 1 or 2 to the score every turn. The first to 20 wins the game.

Technology:
  • C
  • Arduino
  • Hardware:
    • LCD
Automated Fish Feeder

This is an automated fish feeder that uses Arduino to proportionally feed the fish in the tank according to user selection of the number of fish (options range from 1 to 5). The fish feeder takes advantage of an auger-style design in order to bring control to how much fish food is dispensed into the tank.

Technology:
  • C
  • Arduino
  • Hardware:
    • LCD
    • 360 Degree Continuous Rotation Servo
Obstacle Avoiding Robot

This is a obstacle avoiding robot I built that is able to move around in an environment by itself. Initially the robot had a fixed ultrasonic sensor orientation (front) but I placed the ultrasonic sensor above a servo motor to introduce two new orientations for the sensor to view and make a more appropriate decision.

Technology:
  • C
  • Arduino
  • Hardware:
    • HC-SR04 Ultrasonic Sensor
    • SG90 Micro 9g Servo
    • L293D Motor Drive Shield
Live Streaming Robot

This was the next iteration of the obstacle avoiding robot which used cameras and a Raspberry Pi. An Xbox controller would allow control of the robot's movement, including the horizontal tilt of the camera view as it would move the servo the camera is placed on. The visuals would be streamed to a local site using WebRTC and Apache.

Technology:
  • Python
  • Raspberry Pi
  • HTML, CSS, Javascript
  • Apache
  • WebRTC
  • Hardware:
    • Raspberry Pi Camera Module
    • SG90 Micro 9g Servo
    • L298n Dual H-Bridge DC Stepper Motor Controller

Todo App

This is a simple todo app that allows users to enter tasks todo which would then be displayed. They can check off completed tasks as well as update and delete any entered tasks. In addition, they can filter through the tasks in terms of all, completed or pending tasks, making it an easier experience to find the tasks they are looking for. As a bonus there is also a dark mode and light mode option to give users an even better UI experience.

Technology:
  • Angular
  • HTML, CSS, Typescript
  • Github
  • Visual Studio Code
